Financial Fellow is a full-stack personal finance web application built to help users track expenses, plan budgets, and analyze spending patterns over time. The project follows a structured development plan and process flow, emphasizing practical full-stack development, data visualization, and user-centered design.

Features
  
    Expense logging with categorized monthly breakdowns
    
    Budget planning tools to support financial goal tracking
    
    Automated bill reminders to promote timely payments
    
    Interactive monthly and month-to-month spending visualizations

Technical Highlights

    Backend developed with Python and Django, using a modular and scalable architecture
    
    Frontend built with HTML, CSS, and JavaScript, styled with Bootstrap for responsive design
    
    Integrated D3.js to create interactive charts for analyzing monthly expenses and spending trends
    
    Clear separation of concerns between data models, business logic, and presentation layers

Learning Objectives

  This project strengthened experience in:
    
    Full-stack web development and REST-oriented design
    
    Data visualization and client-side interactivity
    
    Translating functional requirements into working features
    
    Building maintainable, user-focused web applications
